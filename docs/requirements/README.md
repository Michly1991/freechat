# FreeChat 项目要求索引

本目录记录维护 FreeChat 时必须长期遵守的项目要求。内容来自当前项目约定、用户明确要求以及长期记忆中的 FreeChat 工作规则。

要求按职责边界拆分：

- `process/`：协作流程、设计确认、文档同步、验证和提交要求。
- `server/`：服务端安全边界、权限、计费、数据一致性和架构拆分要求。
- `client/`：Web/移动端交互、响应式、权限提示和前端架构要求。
- `agent-capabilities/`：Agent Skill/能力文件、脚本、模板和知识库同步要求。

新增或修改 FreeChat 功能时，先检查本目录，再同步 `docs/design/` 中对应系统设计文档。
