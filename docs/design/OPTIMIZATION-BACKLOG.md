
## FreeChat 功能和交互优化清单

### 🔴 未完成/待完善功能

#### 计费相关
1. **钱包充值/提现** - 只有消费和记账，没有充值入口，也没有充值记录页面
   - `/api/wallet/topup` 接口未实现
   - 前端 BillingPanel 只有流水，没有充值按钮/历史
   - 新用户注册赠送 1000 credit，用完会卡住无法继续
   - 建议：增加充值记录 tab，支持手工/admin 充值（对接支付前临时方案）

2. **账单汇总/导出** - 只有流水明细，没有月度汇总/导出
   - 缺少按月份/房间/Agent/模型汇总统计
   - 缺少 CSV/Excel 导出功能
   - 建议：BillingPanel 增加“汇总”tab，按维度展示消费趋势

3. **账单预警/通知** - 余额不足时没有提前通知
   - 余额 < 10 credit 时顶部红点/横幅通知
   - 余额为 0 时强提示引导充值
   - Agent 执行前预检时附带余额不足提醒

4. **分账/结算** - Agent/模型收入只有记账，没有提现/结算
   - 平台收入、Agent 作者收入只有流水，没有提现流程
   - 缺少最低提现额、提现申请、审核、打款记录

#### 访客/分享入口
5. **访客会话列表** - 访客入口已实现，但访客看不到自己的历史会话
   - 目前只有工作组 owner/admin 能在工作组通讯录看到访客会话
   - 访客用户首页看不到“我参与的访客对话”
   - 建议：首页增加“访客会话”分类，展示该用户作为访客加入的对话

6. **访客会话有效期/归档** - 访客对话目前永久保留，没有自动清理
   - 建议：增加“访客会话 30 天自动归档”配置
   - 超过有效期后访客不可见，仅 owner 可见

7. **入口访问控制** - 目前是“登录用户都可访问”，没有白名单/限制
   - 建议：支持“仅指定用户可访问”、“需要审核”、“仅工作组内成员可见”
   - 支持访问次数/频控限制

#### Agent 市场/发布
8. **Agent 上架/市场** - Agent 定价 UI 已实现，但市场列表未完成
   - 目前首页有 MarketSection 占位，没有真实市场列表
   - 缺少搜索/筛选/排序/评分/收藏
   - 缺少“使用此 Agent”创建项目的流程

9. **Agent 版本/发布记录** - Agent 修改没有版本历史
   - 目前每次编辑直接覆盖，没有回滚/历史查看
   - 建议：Agent 保存时自动创建小版本，发布到市场时标记大版本

10. **Agent 运行日志查看** - 只有 token 消耗统计，没有详细运行日志
    - 房间“执行记录”目前只有列表，没有详细日志/错误堆栈
    - 建议：点击运行记录可查看完整日志、输入输出、耗时、错误信息

#### 任务/子任务
11. **任务依赖/阻塞** - 数据结构有 dependencies，但前端没有 UI
    - 任务卡片显示“依赖 N 项”，但不能点击查看具体依赖哪几个
    - 不能在任务面板直接添加/移除依赖
    - 建议：依赖项点击跳转，依赖未完成时任务置灰不可操作

12. **任务附件/评论** - 只能看标题状态，不能添加附件或评论
    - 任务详情抽屉缺少附件上传、评论区
    - 建议：点击任务卡片展开右侧详情抽屉，支持附件/评论/时间线

13. **任务通知** - 任务被分配/评论/完成时，没有消息通知
    - 没有站内信/消息中心
    - 没有“@我的任务”列表

#### 文件/工作区
14. **文件上传/下载** - 后端接口可能有，但前端没有完整的文件浏览器 UI
    - 目前只有 RoomShellChrome 里的新建文件/目录弹窗
    - 没有文件列表、上传按钮、下载、删除、重命名
    - 建议：文件 Tab 增加树形文件浏览器，支持拖拽上传

15. **文件历史/版本** - 文件编辑直接覆盖，没有历史版本
    - 建议：每次保存保留小版本，可查看历史/回滚

16. **大文件处理** - 没有分片上传/断点续传/大小限制提示
    - 目前默认会把大文件塞进去，造成数据库/存储压力
    - 建议：前端限制单文件 < 50MB，超过给出明确提示

#### 语音/ASR/TTS
17. **语音设置** - VoiceSettingsPanel 有表单，但没有实际对接服务
    - ASR/TTS URL 配置后没有测试按钮
    - 房间里没有语音输入/输出 UI
    - 建议：增加“测试语音”按钮，验证配置是否生效

#### 移动端适配
18. **响应式细节优化** - 整体框架已响应式，但很多面板在手机上体验不好
    - 工作组设置/通讯录面板在小屏上左右滚动不好用
    - 任务看板手机上还是多列，应该改成单列表
    - 输入框在手机上被键盘顶起时布局可能错乱
    - 建议：增加移动端专用布局断点 `< sm` 时强制单列

#### 权限/安全
19. **操作日志/审计** - 缺少关键操作审计
    - 删除房间、修改权限、添加/移除成员没有记录
    - 建议：增加 `audit_logs` 表和后台查看入口

20. **密码强度/忘记密码** - 注册只有简单必填，没有强度校验和找回密码
    - 建议：至少 8 位，包含数字字母；增加邮箱/手机号找回密码

### 🟡 交互体验优化

#### 表单/输入
21. **所有表单提交后没有 loading/禁用**
    - 点击按钮后立即禁用并显示 loading，避免重复提交
    - 目前很多接口调用时按钮可重复点击，造成重复创建

22. **表单没有键盘回车提交**
    - 搜索框、新建任务、添加成员等常用操作应支持回车提交
    - 部分地方已有实现，但不全（如新建入口、模型配置）

23. **错误提示不够友好**
    - 接口失败直接展示原始错误信息，很多是技术术语
    - 建议：统一错误拦截，展示用户可理解的文案，附带重试按钮

#### 列表/空状态
24. **空状态页面太简陋**
    - 很多空状态只是一行灰色文字，没有引导操作按钮
    - 例如“暂无流水”应该附带“去创建项目”、“去市场看看”等操作

25. **列表加载更多/分页**
    - 目前大部分列表是全量拉取，数据多了会卡
    - 建议：超过 50 条自动分页或虚拟滚动

#### 反馈/动效
26. **操作成功后没有反馈**
    - 保存配置、添加成员等操作成功后只是消失弹窗，没有 toast 或成功提示
    - 建议：所有写操作完成后给明确的“已保存”/“已添加”反馈

27. **网络慢时没有加载状态**
    - 很多接口调用时没有 loading，用户以为没点到
    - 建议：按钮点击后立即进入 loading，禁用交互

#### 导航/返回
28. **页面没有面包屑/返回指引**
    - 从首页进入房间设置，再进入 Agent 设置，想返回时不知道怎么回去
    - 建议：增加顶部面包屑，或统一左上角返回按钮

#### 搜索
29. **搜索没有防抖**
    - 很多搜索框 onChange 直接调用接口，打字时频繁请求
    - 建议：统一加 300ms 防抖

30. **搜索没有清空按钮**
    - 输入内容后想清空得手动删除，不方便
    - 建议：搜索框有内容时显示 × 按钮

### 🟢 建议优先处理顺序

**P0（影响核心流程）**
1. 钱包充值入口（哪怕手工/admin 充值先顶上去）
2. 余额不足时的强提示和引导
3. 访客看不到自己历史会话的问题

**P1（体验明显提升）**
4. 所有表单按钮 loading + 防重复提交
5. 空状态引导操作
6. 搜索防抖和清空按钮
7. 任务详情抽屉和评论

**P2（中长期优化）**
8. 文件浏览器完整 UI
9. Agent 市场和版本管理
10. 语音通话完整对接
11. 操作审计日志
12. 月度账单汇总/导出
